# Portfolio Website

Personal portfolio website of Akash Singh Bisht, built with Next.js, React, Tailwind CSS, and deployed on Vercel.

## Live Demo

https://akashportfolio-rosy.vercel.app/

## Tech Stack

- Next.js
- React.js
- Tailwind CSS
- JavaScript
- lucide-react
- Vercel

## Sections

- Hero
- Projects
- Skills
- Experience & Education
- Contact

## Run Locally

```bash
git clone https://github.com/AKASH-18/Portfolio-Website.git
cd Portfolio-Website
npm install
npm run dev
```
